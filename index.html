"Maar waar kom je echt vandaan?"

Het antwoord op de beruchte vraag

Hoi, mijn naam is SHANDRA. In mijn leven heb ik een handje vol keer de vraag gekregen: "Waar kom je vandaan?". Ik antwoord dan: "Nederland" maar ik krijg daarna snel de vraag: "Maar waar kom je echt vandaan?" Ik wist nooit goed hoe ik hierop moest antwoorden.

Ik wist wel deels waar mijn grootouders vandaan kwamen maar ik wilde graag precies weten hoe het zat dus heb ik een DNA test gedaan. Misschien vind je dit een beetje apart maar afkomst is voor veel mensen een belangrijk deel van hun identiteit. Ik wilde dit graag in kaart brengen en heb daarom deze website gemaakt. Scroll verder voor een stukje van mijn geschiedenis. Veel plezier!
